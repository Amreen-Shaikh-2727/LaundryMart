# LaundryMart

LaundryMart is a responsive laundry service web application built using HTML, CSS, and JavaScript. The project provides users with a clean and modern interface to browse laundry services, add items, manage bookings, and view selected services before placing an order.

## Live Demo

https://amreen-shaikh-2727.github.io/LaundryMart/

---

## Features

* Responsive and modern user interface
* Service cards with item selection functionality
* Add and remove laundry items
* Dynamic cart management
* Real-time booking updates
* Interactive buttons and user feedback
* Mobile-friendly design
* Clean and organized code structure

## Installation

### Clone the Repository

```bash
git clone https://github.com/amreen-shaikh-2727/LaundryMart.git
```

### Navigate to Project Directory

```bash
cd LaundryMart
```

### Run the Project

Since this is a frontend project, simply open the `index.html` file in your browser.

Alternatively, use VS Code Live Server:

1. Open the project in Visual Studio Code.
2. Install the Live Server extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.

---

## Usage

1. Browse available laundry services.
2. Select items to be added.
3. View selected items in the cart section.
4. Remove items if necessary.
5. Proceed with booking when items are selected.

---

## Key Functionalities

### Service Management

* Add laundry items to the booking list.
* Remove selected items instantly.

### Booking Validation

* Booking button remains disabled when no items are selected.
* Prevents empty bookings.

### Dynamic User Interface

* Real-time updates without page refresh.
* Interactive button states for better user experience.
